# University
Final Project for Integrative Programming and Technologies

## Description
This project is a web-based system that manages student accounts, roles, and approvals.

## Features
- User registration (student, cashier, finance, dean)
- Admin approval system
- Login and authentication
- Role-based access

## Installation
1. Download files
2. Import the database schoold_db and edata in phpMyAdmin
3. Configure db_connection.php
4. Run on localhost (XAMPP)

## Usage
- Register as different roles
- Wait for admin approval
- Login and access dashboard
